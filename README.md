# Phitron Projects

This repository contains Jupyter Notebook assignments and exam solutions for Python-based coursework, along with the datasets used in the notebooks.

## Repository Structure

```text
Assignment/
  AI_Programming_With_Python_Assignment_01_(abdurrahmansoftw_gmail_com).ipynb
  AI_Programming_With_Python_Final_Exam_Question.ipynb
  midterm_solutions.ipynb
Assignment-2/
  index.ipynb
data/
  masters_data_science_performance.csv
  titanic_data_updated.csv
```

## What’s Inside

- `Assignment/` contains notebook-based solutions and question sets for the AI Programming With Python course.
- `Assignment-2/` contains an additional notebook named `index.ipynb`.
- `data/` contains CSV datasets referenced by the notebooks.

## Datasets

- `masters_data_science_performance.csv`
- `titanic_data_updated.csv`

## Requirements

- Python 3
- Jupyter Notebook or JupyterLab

## How To Use

1. Open the repository in your editor or Jupyter environment.
2. Launch any notebook from the `Assignment/` or `Assignment-2/` folders.
3. Make sure the `data/` folder stays in place so the notebooks can load the CSV files correctly.
